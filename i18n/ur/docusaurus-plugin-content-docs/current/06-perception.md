# نیورل ڈائنامکس کا تعارف (Introduction)

# روبوٹک ادراک (Robotic Perception)

روبوٹ اپنے ماحول کو سینسرز کے ذریعے دیکھتا اور سمجھتا ہے。

* **LiDAR:** تھری ڈی (3D) میپنگ کے لیے لیزر کا استعمال کرنا。
* **کمپیوٹر ویژن:** کیمروں کے ذریعے راستے میں آنے والی چیزوں کو پہچاننا。
* **ڈیپتھ سینسنگ (Depth Sensing):** چیزوں کا روبوٹ سے فاصلہ معلوم کرنا。

# تھری ڈی (3D) دنیا کو سمجھنا

فزیکل اے آئی متحرک ماحول میں محفوظ طریقے سے کام کرنے کے لیے اعلیٰ معیار کے ادراک (Perception) پر انحصار کرتی ہے。 یہ باب اس بات کی وضاحت کرتا ہے کہ روبوٹ اپنے گردونواح کو کس طرح "دیکھتے" اور "محسوس" کرتے ہیں۔

## 6.1 ملٹی موڈل سینسر سویٹ (Sensor Suite)
مضبوط خودمختاری حاصل کرنے کے لیے، روبوٹ **سینسر فیوژن** (Sensor Fusion) کا استعمال کرتے ہیں، جو مختلف ذرائع سے حاصل کردہ ڈیٹا کو یکجا کرتا ہے:
* **ایکٹیو سینسنگ (LiDAR):** لائٹ ڈیٹیکشن اینڈ رینجنگ۔ یہ سینسرز کمرے کا تھری ڈی "پوائنٹ کلاؤڈ" بنانے کے لیے ہر سیکنڈ میں لاکھوں لیزر پلس خارج کرتے ہیں。
* **پیسو سینسنگ (RGB-D):** Intel RealSense جیسے ڈیپتھ کیمرے ہر پکسل کے لیے رنگین تصاویر اور فاصلے کا ڈیٹا فراہم کرتے ہیں。
* **ٹیکٹائل فیڈ بیک (Tactile Feedback):** روبوٹ کے ہاتھوں میں موجود پریشر سینسرز جو اسے یہ "محسوس" کرنے دیتے ہیں کہ آیا کوئی چیز پھسل رہی ہے。

## 6.2 سیمنٹک سیگمنٹیشن اور سین پارسنگ
عام آبجیکٹ ڈیٹیکشن کے برعکس، **سیمنٹک سیگمنٹیشن** (Semantic Segmentation) روبوٹ کو ہر ایک پکسل کی درجہ بندی کرنے کی اجازت دیتی ہے:
* **ساکن رکاوٹیں (Static Obstacles):** دیواریں، فرش اور بھاری فرنیچر。
* **متحرک رکاوٹیں (Dynamic Obstacles):** انسان، پالتو جانور اور دیگر متحرک روبوٹس。
* **قابلِ عبور سطحیں (Traversable Surfaces):** وہ جگہیں جہاں روبوٹ کی ٹانگیں یا پہیے محفوظ طریقے سے حرکت کر سکتے ہیں。

## 6.3 تکنیکی نفاذ: رکاوٹوں کی شناخت (Obstacle Detection)

```python
import numpy as np

def analyze_depth_buffer(depth_map, safety_margin=0.45):
    """
    ڈیپتھ ڈیٹا کا تجزیہ کرتا ہے تاکہ اگر کوئی چیز حفاظتی حد کے اندر ہو 
    تو ایمرجنسی اسٹاپ کو متحرک کیا جا سکے۔
    """
    closest_point = np.min(depth_map)
    if closest_point < safety_margin:
        return f"انتہائی اہم اسٹاپ: {closest_point:.2f} میٹر پر رکاوٹ کی شناخت ہوئی ہے"
    return "راستہ صاف ہے: نارمل آپریشن جاری رکھیں"

# مصنوعی 10x10 ڈیپتھ فریم
simulated_view = np.random.uniform(0.3, 5.0, (10, 10))
print(analyze_depth_buffer(simulated_view))
