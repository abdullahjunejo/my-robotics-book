# Robot Operating System 2 (ROS 2)

ROS 2 robotics industry ka standard software hai jo saare parts ko aapas mein jorta hai.

* **Nodes:** Chote software programs jo alag alag kaam karte hain.
* **Topics:** Wo rasta jis par data (jaise camera feed) travel karta hai.
* **DDS:** Secure aur tez communication ka nizam.


#  The Backbone of Robotic Communication

The Robot Operating System (ROS 2) is the industry-standard middleware that facilitates communication between the robot's brain and its hardware.

## 8.1 The Computational Graph
Every function in a robot is a **Node**. Nodes communicate through:
* **Topics:** Constant streams of data (e.g., LiDAR scans at 30Hz).
* **Services:** Request/Response for specific tasks (e.g., "Calibrate Sensors").
* **Actions:** Goal-oriented tasks with feedback (e.g., "Walk 5 meters").



## 8.2 Data Distribution Service (DDS)
ROS 2 uses DDS to ensure **Quality of Service (QoS)**. This is critical because balance-critical data must always be prioritized over non-critical logging data.
